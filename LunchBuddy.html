Results

I was able to create a design solution that incorporated the following:

Information Architecture

  • One Window Drilldown - to help user focus on one page of information at a time
  • Extras on Demand - hiding the menu for user until they need the additional options

Navigation and Landmarks

  • Clear Entry Points - to help user get to Funding and Lunch Menu quicker
  • Sequence map - to help user know where they are during new user creation
  • Escape Hatch - to help user back out or go back to where they were

Content Organization

  • Center stage - focus user on the most important content
  • Titling - Titles on pages and card provide users information of where they are and what they are looking at
  • Responsive Disclosure - immediately on application load, user can sign in or sign up

User Inputs

  • Input Hints - text fields have placeholder text with either direction or sample information for user to understand what is needed to be entered
  • Microinteractions - allows users to complete add favorite meals to a child with a single action

May take up to 30 seconds to load.

Lessons Learned

This project has allowed me to understand more about UI design patterns and how it affects the user experience. I was able to experiment with micro interactions which is critical in bringing simplicity and delight into a design solution.
